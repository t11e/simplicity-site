Namespace $.ui.simplicityYahooMapBoundsTracker

A Yahoo! map.

An invisible jquery ui widget which tracks changes in a Yahoo! Map's bounds. Whenever the map bound change, a simplicitybingmapboundstrackerbounds event is triggered with ui.bounds being the normalized map bounds.
Defined in: ui.simplicityYahooMapBoundsTracker.js.
