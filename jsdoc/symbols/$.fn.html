<static> $.fn.simplicityFromState(state, triggerChange, debug)
Plugin to apply state to input elements. Apply this to input elements to change their state based on a state Object.
Defined in: fn.simplicityFromState.js. 
  var state = {
    "shape": "circle",
    "size": ["small", "medium"]
  };
  $(':input').simplicityFromState(state);
Parameters:
state
The target Object to extract the state to.
triggerChange
Optional argument, when true invokes change() on any elements that were changed.
debug
Optional argument, when true enables logging via console.log.
<static> $.fn.simplicityToState(state, trimValue, ignoreEmptyValues)
Plugin to extract state from input elements. Apply this to input elements to extract their state into a dictionary.
Defined in: fn.simplicityToState.js. 
  var state = {};
  $(':input').simplicityToState(state);
  console.log(state);
Parameters:
state
The target Object to extract the state to.
trimValue
The state value will be trimmed if this is true.
ignoreEmptyValues
The state value will be ignored if it is empty and this is true. Useful for applying state on initial page load with multiple inputs mapped to the same fields.
